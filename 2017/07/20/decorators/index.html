sitemesh布局和修饰框架

2017-07-20
Java

#简述
SiteMesh是一个用来在JSP中实现页面布局和装饰的框架组件，能够较容易实现页面中动态内容和
静态装饰外观的分离。

#说明
1.首先通过Maven来管理sitemesh

2.在web.xml中配置

3.decorators.xml
装饰器配置文件与web.xml同级放在WEB-INF下。defaultdir为布局文件，pattern表示作用域
