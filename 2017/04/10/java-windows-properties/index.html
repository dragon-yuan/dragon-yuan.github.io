通过JAVA环境使用sigar监听Windows系统属性

2017-04-10
Java

使用sigar开源组件，获取当前系统各项属性
1.下载sigar.jar
2.将sigar-amd64-winnt.dll，sigar-x86-winnt.dll，sigar-x86-winnt.lib
复制到JDK的bin目录下，使sigar拥有运行依赖。
