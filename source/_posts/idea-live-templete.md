类注释：
className - className()
date - date()
time - time()
```java
/**
 * <p>Title: $className$.$END$ </p>
 * <p>Description TODO </p>
 * @author dragon
 * @date $date$ $time$
 */ 
```

方法注释：
methodName - methodName()
param - methodParameters
date - date()
time - time()
return - methodRetureType()
```java
/**
 * <p>Title: $methodName$.$END$ </p>
 * <p>Description </p>
 * @param $param$
 * @author dragon
 * @date $date$ $time$
 * @return $return$
 */
```
